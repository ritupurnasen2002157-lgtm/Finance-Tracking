# Personal Finance Tracker (Expense & Budget Manager)

A responsive personal finance tracker web app built with **HTML, CSS, and JavaScript**.

## Features
- Add / Edit / Delete expenses (Title, Amount, Category, Date)
- Set a monthly budget and track remaining amount
- Data persistence using **LocalStorage**
- Category filtering + search by title/category
- Basic charts (Chart.js)
- Dark/Light mode toggle
- Export visible expenses to CSV

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Chart.js (via CDN)

## How to Run
1. Open `index.html` in a browser.
   - (Optional) Use Live Server in VS Code.
2. Start adding expenses and set your budget.

## LocalStorage Keys
- `pft_expenses_v1` (expenses)
- `pft_budget_v1` (monthly budget)
- `pft_theme_v1` (theme)

## Notes (Demo / Internship)
All data stays in your browser and remains after refresh.

## Deployment
You can deploy directly on:
- **GitHub Pages**
- **Netlify**
- **Vercel**
